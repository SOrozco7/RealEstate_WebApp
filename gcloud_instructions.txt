Para probar en localhost:

0. Ir desde la terminal a la carpeta del proyecto
1. Ejecutar lo siguiente en la terminal
    
    dev_appserver.py .

Para montar en Google Cloud

0.- Ir desde la terminal a la carpeta del proyecto
1.- gcloud projects create PROJECT_NAME
2.- gcloud config set project PROJECT_NAME
3.- gcloud app create
4.- gcloud app deploy
